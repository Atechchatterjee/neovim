# Neovim Configuration

This neovim configurations has been taken from [**nvim-lua/kickstart**](https://github.com/nvim-lua/kickstart.nvim). I have added some extra plugins, keybinds and configurations to my use-case.

**NOTE:** The ***old*** directory contains my earlier version of my nvim configuration.
*(This folder is temporary and is used only for reference)*

The plugins that I have added:

1. [Telescope](https://github.com/nvim-telescope/telescope.nvim)
2. [Harpoon](https://github.com/ThePrimeagen/harpoon)
3. [Nvim-tree](https://github.com/nvim-tree/nvim-tree.lua)
4. Themes: [solarized](https://github.com/ishan9299/nvim-solarized-lua), [nightfly](https://github.com/bluz71/vim-nightfly-colors)
5. [Devicons](https://github.com/nvim-tree/nvim-web-devicons)
6. [Autopairs](https://github.com/windwp/nvim-autopairs)
7. [Dressing](https://github.com/stevearc/dressing.nvim)
